#Ybook

Ybook est une web-app destiné au étudiant leur permettant de pouvoir discuter sur une plateforme commune

## Installation

Clone le projet

```bash 
git clone git@github.com:Ybook-Riri-fifi-loulou/Ybook-back.git
```

initialise le projet

```bash
npm install
```

generate prisma
```bash
npx prisma generate
```

start le projet

```bash
npm run start
```
